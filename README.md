# Orbit — Your AI Habit Tracker 🌌

Orbit is a full-stack web application that helps users build consistent routines using deterministic logs, personalized 3-day recovery configurations, and an automated AI coaching engine powered by the latest `gemini-3.5-flash` model.

## 🚀 Key Features
* **AI Habit Coach:** Generates periodic insights, tailored recovery strategies for broken streaks, and interactive chat assistance grounded tightly in localized metrics.
* **Deterministic Tracking Engine:** Simulates long-term realistic habit behavior using unique mathematical variance modeling (`Math.sin` distribution).
* **Robust Fallback Safeguards:** Built-in validation error catching and hardcoded gracefully degrading suggestion fallbacks if API limits are reached.

## 🛠️ Tech Stack
* **Frontend:** React, Tailwind CSS, Lucide Icons, Vite
* **Backend:** Node.js, Express, Mongoose (MongoDB)
* **AI Pipeline:** Google GenAI SDK (`gemini-3.5-flash`)
